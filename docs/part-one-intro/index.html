介绍

通过学习本教程，你将学习如何快速使用Python，Postgres，以及Flask*Web框架在*Docker中构建一个RESTful API开发环境。在学习完如何在本地实现后，我们将学习部署到Amazon EC2实例中。

我们将通过测试驱动开发模式(TDD)过程来实现，在实现前我们先会编写单元测试。

本系列使用如下工具和技术：

Part 1

  1. Python v3.6.1
  2. Flask v0.12.2
  3. Flask-Script v2.0.5
  4. Flask-SQLAlchemy v2.2
  5. Flask-Testing v0.6.2
  6. Gunicorn v19.7.1
  7. Nginx v1.13.0
  8. Docker v17.03.1-ce
  9. Docker Compose v1.11.2
  10. Docker Machine v0.10.0

Part 2

  1. Coverage.py v4.4.1
  2. Node v7.10.0
  3. NPM v4.2.0
  4. Create React App v1.3.0
  5. Axios v0.16.2
  6. Flask-CORS v3.0.2

在开始前，你应该熟悉以下主题，参考如下获取更多信息：

Topic Resource
Docker Get started with Docker
Docker Compose Get started with Docker Compose
Docker Machine Docker Machine Overview
Flask Real Python

目标

在本章之后，你应该可以……

  1. 使用Flask开发RESTful API
  2. 测试驱动开发实践
  3. 在本地配置Docker，Docker Compose和Docker Machine并执行服务
  4. 使用volumes挂载代码到容器里
  5. 在Docker容器里执行单元测试和集成测试
  6. 服务在不同容器间相互通信
  7. Python和Flask在容器中执行
  8. 在Amazon EC2实例安装Flask, Nginx和Gunicorn

App

查看运行在EC2的Demo示例 - http://54.89.0.180/

你也可以测试如下地址：

Endpoint HTTP Method CRUD Method Result
/users GET READ get all users
/users/:id GET READ get single user
/users POST CREATE add a user

该应用运行在三个容器 - Flask，Postgres和Nginx。在第一部分末，你将完成该应用并且部署。我们会在接下部分添加验证和其他服务。
